Se genera un JAR simple construido con Maven. En donde se utiliza el jar "Generador_Datos_Desafio_Tres-1.0.0.jar"
como libreria externa y es configurada en el archivo porn.xml.
El archivo de salida configurado es el tipo "csv".

Las tecnologías utilizadas son Java 8 y la libreria GSON, JUnit y el jar externo configurado como desafio3.

Para la compilacion basta con importar el proyecto como maven, realizar maven update (de ser automatico) y
ejecutar la clase principal "App.java".
